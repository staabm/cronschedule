cronschedule
============

My first Silex experiments. App just takes a cronexpression and tells you when the next runs will be scheduled.

Example
=======

Input 
```
"*/19 * * * *"
```

Output

```
2013-12-13 10:57:00
2013-12-13 11:00:00
2013-12-13 11:19:00
2013-12-13 11:38:00
2013-12-13 11:57:00
2013-12-13 12:00:00
2013-12-13 12:19:00
2013-12-13 12:38:00
2013-12-13 12:57:00
2013-12-13 13:00:00
```
